---

##  **MetaMask 지갑에 USDT 토큰 추가하기**

1. MetaMask 지갑을 엽니다 (웹 확장 또는 모바일 앱)
2. 네트워크를 선택합니다 (예: Ethereum, Arbitrum, Sepolia 등)
3. **토큰 탭** > **"토큰 가져오기 (Import Token)"** 클릭
4. 아래의 **USDT 컨트랙트 주소**를 복사해 입력
5. 자동으로 심볼 및 소수점 정보가 채워지고, **확인 후 저장**합니다

##  **Phantom 지갑에 USDT 토큰 추가하기**

1. Phantom 지갑을 엽니다 (웹 확장 또는 모바일 앱)
2. 우측 상단 메뉴 > **"Manage Token List" (토큰 목록 관리)** 클릭
3. EVM 네트워크 선택 (예: Ethereum, Polygon, Base 등)
4. **"Add custom token"** 또는 **"Import Token"** 선택
5. 아래 컨트랙트 주소 입력
6. 자동으로 토큰 정보가 채워지고 저장됩니다

##  **Rabby 지갑에 USDT 토큰 추가하기**

1. Rabby 지갑을 엽니다 (웹 확장 또는 모바일 앱)
2. 네트워크를 선택합니다 (예: Ethereum, Arbitrum, Polygon 등)
3. **토큰은 자동으로 표시**되므로 별도로 토큰을 수동으로 추가할 필요 없이 사용 가능합니다.

> 📝 Rabby는 사용자가 주소를 가져오기만 하면 **자동으로** 네트워크에 맞는 **모든 토큰을 표시**해 주기 때문에 추가적인 설정 없이 바로 사용할 수 있습니다.
