MetaMask, Rabby와 같은 지갑은 기본적으로 Ethereum 메인넷만 탑재되어 있으므로 필요한 네트워크는 수동 또는 자동으로 추가해야 합니다.'

## 테스트넷 (Testnet)

테스트 환경에서는 아래 네트워크들을 사용할 수 있습니다.

| 네트워크 | 설명 | 추가 방법 |
|----------|------|-----------|
| **Sepolia** | Ethereum 공식 테스트넷 | ✅ 기본탑재 |
| **Arbitrum Sepolia** | Arbitrum 테스트넷 | [Chainlist에서 추가](https://chainlist.org/?search=arbitrum+sepolia&testnets=true) |
| **Base Sepolia** | Coinbase의 L2 테스트넷 | [Chainlist에서 추가](https://chainlist.org/chain/84532) |
| **Amoy (Polygon)** | Polygon 테스트넷 | [Chainlist에서 추가](https://chainlist.org/?search=amoy&testnets=true) |

> 💡 [Chainlist.org](https://chainlist.org) 에서는 다양한 EVM 체인을 쉽게 지갑에 추가할 수 있습니다.

## 메인넷 (Mainnet)

실제 결제와 거래를 위한 메인넷 목록입니다.

| 네트워크 | 설명 | 추가 방법 |
|----------|------|-----------|
| **Ethereum** | 이더리움 메인넷 | ✅ 기본탑재 |
| **Arbitrum One** | Arbitrum 메인 체인 | Chainlist 또는 지갑 내 네트워크 추가 |
| **Base Mainnet** | Coinbase 기반 L2 | Chainlist 또는 수동 설정 |
| **Polygon (POS)** | Polygon 메인 체인 | Chainlist 또는 지갑 내 네트워크 추가 |

---

## 🔗 Chainlist 사용법

1. [Chainlist.org](https://chainlist.org) 접속  
2. 상단 검색창에 네트워크 이름 입력 (예: `arbitrum sepolia`)  
3. **Connect Wallet** 클릭  
4. **Add to Wallet** 버튼 클릭 → 지갑에서 자동 추가 확인
