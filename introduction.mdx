## 🦊 MetaMask (이더리움 기반)

**MetaMask**는 Ethereum 및 EVM 기반 네트워크에서 가장 널리 사용되는 지갑입니다.

### 설치 방법
1. [MetaMask 공식 웹사이트](https://metamask.io/)에 접속합니다.
2. 사용하는 브라우저에 확장 프로그램을 설치합니다.
3. 지갑을 새로 생성하거나, 기존 지갑을 복구합니다.
4. **비밀 복구 구문(Secret Recovery Phrase)** 은 반드시 안전한 장소에 백업하세요.

## 🦝 Rabby Wallet (EVM 멀티체인 지갑)

**Rabby**는 MetaMask의 대안으로 주목받는 **EVM 기반 멀티체인 지갑**입니다.  
자동 체인 감지, 보안 기능 강화, 직관적인 UI 등을 제공합니다.

### 설치 방법
1. [Rabby 공식 웹사이트](https://rabby.io/)에 접속합니다.
2. Chrome 브라우저 확장 프로그램을 설치합니다.
3. 지갑을 새로 생성하거나, 기존 지갑을 복구합니다..
4. 복구 구문은 안전하게 보관하세요.

## 🐧 Phantom (솔라나 기반)

**Phantom**은 Solana 네트워크 전용으로 개발된 지갑입니다.  
Solana 기반의 DApp, NFT, 토큰 전송 등에 최적화되어 있습니다.

### 설치 방법
1. [Phantom 공식 웹사이트](https://phantom.app/)에 접속합니다.
2. 브라우저 확장 프로그램 또는 모바일 앱을 설치합니다.
3. 새 지갑을 생성하거나 기존 지갑을 복구합니다.
4. 복구 구문은 안전하게 보관하세요.
